# Site Juju

Repositório do site. O front-end está em **Angular** na pasta `sitejuju-angular/`.

## Ver o site na web (GitHub Pages)

1. No GitHub, abra o repositório **nickcintra/sitejuju**.
2. Vá em **Settings** → **Pages** (menu lateral).
3. Em **Build and deployment** → **Source**, escolha **GitHub Actions**.
4. Faça um **push** para a branch `master` (ou rode o workflow manualmente em **Actions**).

Depois do primeiro deploy, o site fica em:

**https://nickcintra.github.io/sitejuju/**

Qualquer pessoa pode acessar esse link (celular, PC, etc.) sem rodar nada localmente.
